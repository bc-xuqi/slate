# 1. 概览

壹收款基于BeeCloud SDK，是BeeCloud在后者基础上专门开发出的一个轻量级版本。它只需要短短一句代码即可使用，并且内嵌了一套风格简约，和绝大部分应用契合度较高的支付页面和完成页面。这个功能对于入门级开发者来说，不但降低了开发成本，还能够优化用户在支付流程当中的体验。
